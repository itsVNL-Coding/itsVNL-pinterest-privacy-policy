Privacy Policy

Last updated: November 18th, 2025

This Privacy Policy explains how my personal app (“the App”) handles data when connecting to Pinterest using the official Pinterest API. The App is used for personal purposes only, through n8n workflows, to create Pins on my own Pinterest account. No other users’ accounts are accessed, and no data is shared with third parties.

1. What the App Does

The App allows me to:

  • Create Pins on my Pinterest account using n8n workflows.
  • Automate Pin creation for my own content.

All actions require my explicit authorization through Pinterest OAuth, and the App does not perform any actions on other users’ accounts.

2. How Pinterest Data Is Accessed

The App uses Pinterest’s OAuth 2.0 flow. I must explicitly authorize the App to access my Pinterest account. The App does not request or store Pinterest passwords, session cookies, or login credentials.

Once authorized, the App may access:

  • My public profile information
  • My boards (to allow me to choose where a Pin will be published)
  • Posting capabilities for creating Pins (only when I initiate the action)

The App does not modify existing Pins, delete content, follow/unfollow other users, or take any actions I did not directly trigger.

3. Data Storage

The App:

  • Does not permanently store Pinterest data retrieved through the API.
  • Only processes Pinterest data temporarily to complete my actions.
  • Does not combine Pinterest data with other users’ accounts or external platforms.

4. User Initiated Actions Only

All actions are performed only after my explicit authorization:

  • I must select or approve every Pin to be created.
  • No automated following, unfollowing, liking, saving, or commenting occurs.
  • Automation through n8n is configured and triggered by me only.

5. How I Use My Information

Pinterest data is used only to create Pins on my own account and to display my boards for selection. The App does not use Pinterest data for advertising, competitor research, or share it with any third parties.

6. Keeping My Data Secure

  • OAuth tokens are stored securely.
  • Unauthorized access to API credentials is prevented.
  • API credentials are not shared with any third party.

7. Revoking Access

I may revoke the App’s access to my Pinterest account at any time via Pinterest’s settings:

Settings → Security & Logins → Apps

Once revoked, the App loses all access to Pinterest data and cannot perform any actions.
